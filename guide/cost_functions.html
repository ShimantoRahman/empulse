Boosting Algorithms With Cost Functions

Empulse provides a number of cost functions for use with the xgboost.XGBClassifier. The empulse.models module defines the empulse.models.B2BoostClassifier as a convenience, but you can use any of the cost functions.

For example, to use the empulse.metrics.make_objective_acquisition() with xgboost, you can do the following:

import xgboost as xgb
from empulse.metrics import make_objective_acquisition

# create the cost function
cost_function = make_objective_acquisition()

# create the xgboost classifier
clf = xgb.XGBClassifier(objective=cost_function, n_estimators=100, max_depth=3)

# fit the model
clf.fit(X_train, y_train)

# predict
y_pred = clf.predict(X_test)

# predict probabilities
y_pred_proba = clf.predict_proba(X_test)
